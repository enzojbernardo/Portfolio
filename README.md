# Portfolio

This repository is the starting point for a personal portfolio built with React, TypeScript, and Vite.

## What is here

- `scripts/run.sh` starts the app in development mode with `npm run dev`.
- `docs/git-guide.md` explains the basic git workflow.
- `docs/npm-guide.md` explains the basic npm workflow.
- `docs/architecture.md` describes the intended page structure for the portfolio.

## How to run

Install dependencies first:

```bash
npm install
```

Start the app:

```bash
./scripts/run.sh
```

If you prefer, you can run the same dev command directly:

```bash
npm run dev
```

## Beginner flow

1. Read `docs/npm-guide.md` to understand installation and scripts.
2. Read `docs/git-guide.md` to understand how to save and share work.
3. Read `docs/architecture.md` to understand the planned page sections.

## Notes

This repo is currently a scaffold. The docs define the intended structure, but the actual React app still needs to be built.
